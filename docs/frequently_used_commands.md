# Frequently Used Commands

## Docker

```
docker-compose images
docker-compose ps
docker-compose build --no-cache
docker-compose up -d
docker-compose exec app bash
docker-compose stop
docker-compose down --rmi all --volumes
```

## Jupyter Notebook

```
jupyter server list
jupyter notebook --ip=127.0.0.1 --port=8888 --no-browser --allow-root
```
