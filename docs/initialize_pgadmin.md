# Initialize pgAdmin

## pgAdmin

```
// ログイン
http://localhost:5050
admin@example.com
password

// 初期設定
1.「新しいサーバを追加」をクリックする。
2.「一般」タブの「名前」に任意の名前を設定する。
3.「接続」タブの「ホスト名/アドレス」に「postgres」を入力する。
4.「ポート番号」に「5432」を入力する。
5.「管理用データベース」に「postgres」を入力する。
6.「ユーザ名」に「admin」を入力する。
7.「パスワード」に「password」を入力する。
8.「パスワードを保存」をオンにする。（任意）
9.「保存」をクリックする。
```
